# suda

#### 介绍
suda是一款基于SpringMVC(Servlet)的轻量级IO安全插件。它具有稳定性、易用性、灵活性等特点，能够有效的防范XSS攻击、 
SQL注入攻击以及识别出潜在的恶意文件（篡改文件后缀名）。此外还增加了字符串去空格和文件后缀名黑名单等功能。
